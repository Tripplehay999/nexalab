# NexaLab Studios Website

A multi-page static site for NexaLab Studios.

## Pages

- `index.html` — home overview and navigation hub.
- `what-we-do.html` — services + monthly retainer plans.
- `who-we-serve.html` — target client profiles.
- `philosophy.html` — operating philosophy.
- `contact.html` — contact CTA.

## Run locally

```bash
python3 -m http.server 4173
```

Then open <http://localhost:4173>.
